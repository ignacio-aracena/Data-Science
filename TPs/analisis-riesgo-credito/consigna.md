## Consigna

Resultado de solicitudes de préstamos en un banco alemán 
El trabajo consiste en analizar la base de datos de solicitudes de préstamos para un banco alemán, para más adelante poder construir un modelo que reproduzca el criterio utilizado por el banco para aceptar o rechazar dichas solicitudes.
Se adjunta un decodificador de la base de datos en el archivo german_clean.doc para que puedan entender el significado de cada variable codificada, y utilizarlo en su análisis.
Realicen el estudio más exhaustivo posible desde el análisis de datos que permita construir un modelo predictivo de la aceptación o rechazo de solicitudes.
El desafío en este dataset es que la muestra es relativamente pequeña para la cantidad de variables, por lo que muchas de ellas serán poco representativas (bajo tamaño muestral) o poco relevantes (bajo information value, baja correlación contra la aceptación o rechazo) .
Aplique las herramientas de visualización y análisis de datos que conoce, tratando de explicar con cada técnica para qué la usa, que espera obtener de cada una de ellas, que resultados obtuvo y qué conclusiones se desprenden. Si no es capaz de responder alguna de estas preguntas para la herramienta que usó, haga el esfuerzo de poder elaborar una respuesta o evite usar dicha herramienta.
Se adjunta un trabajo sobre un dataset similar para que se familiaricen con las variables y las métricas que se emplean.
